# ဓာတ်ပုံများထည့်သွင်းခြင်း

## စည်းမျဉ်းများ:
1. ဓာတ်ပုံအရွယ်အစား: 400x250 pixels (အကြံပြုထားသော)
2. ဖိုင် format: JPG, PNG, WebP
3. အမည်ပေးနည်း: သတင်းခေါင်းစဉ်_ရက်စွဲ.jpg
   ဥပမာ: စာအုပ်စင်အသစ်_20260115.jpg

## ထည့်သွင်းနည်း:
1. ဓာတ်ပုံများကို ဤ folder တွင်ထည့်ပါ
2. Admin panel တွင် ဓာတ်ပုံ URL ကို ဤကဲ့သို့ ရေးပါ:
   ../images/သင့်ဓာတ်ပုံဖိုင်.jpg
